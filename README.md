# Bob D Black Car Service

Modern single-page marketing site for a North New Jersey executive black car service. Built with semantic HTML, modular CSS, and lightweight JavaScript enhancements for AI-inspired ride planning interactions.

## Getting started

Open `index.html` in your preferred browser to view the experience. All assets are local and require no build step.

## Key features

- Hero section with strong call to action and social proof metrics
- Detailed sections covering services, fleet, coverage, testimonials, and contact details
- Responsive navigation with mobile-friendly menu toggle
- AI concierge form that simulates conversational scheduling responses
- SEO-friendly metadata and copy tailored for North New Jersey black car service searches
